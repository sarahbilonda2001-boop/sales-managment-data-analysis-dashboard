# Sales Management Dashboard

## Overview
This project is a sales management dashboard designed to analyze sales performance across different dimensions.

## Tools Used
- Power BI (or Excel)
- Data analysis
- Data cleaning
- Data visualization

## Data Cleaning
Before building the dashboard, the data was cleaned by:
- Removing duplicates  
- Handling missing values  
- Formatting data types  
- Organizing the dataset for analysis  

## Dashboard Features
The dashboard includes the following analysis:

- Sales by day and store manager  
- Total price and quantity by product  
- Quantity by product and time of day  
- Quantity by payment type  
- Payment type by day of the week  

## Purpose
This is my first data analytics project.  
The goal is to practice data cleaning, analysis, and visualization.
